# NZDRA Modernized Web Platform

A modern, polished recreation of the NZDRA website, designed to work in parallel with DragCentral.

## Vision
Modernizing the digital presence of the New Zealand Drag Racing Association (NZDRA) with a focus on usability, real-time data, and community engagement.

## Core Features
- **Central Calendar:** Shared schedule between NZDRA and DragCentral.
- **Smart Race Results:** Filterable results with custom date ranges and easy-to-read layouts.
- **AI-Powered Rulebook:** A searchable database of race rules with an integrated AI assistant to answer technical questions.
- **Admin Portals:** Specialized access for track officials to update race data and for NZDRA to manage rules.
- **Local Track Management:** Empowering individual tracks to manage their own local calendars.

## Technical Foundation
- Web-based registration portal (Mobile optimized).
- Automated data sync with Portatree Paradox databases.
- AI assistant integration for technical and results-based queries.
